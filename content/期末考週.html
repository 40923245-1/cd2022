期末考週

1.利用老師目錄中的 uArm 機械手臂零件, 修改 "nx12絞鍊"與 "front joint" 零件的外型, 使能將 "Triangular connecting rod", "left link" 與 "end link" 等零組件, 與 "right crank" 與 "right link" 的組件左右互換

2.各組員在完成題目 1. 的任務後, 在個人 cd2022 網頁中, 新增一個 H1 格式標題為 "期末考週" 的頁面, 並將題目 1. 的零組件繪圖過程, 以 Wink 影片展示過程細節說明, 並在頁面中說明所採用的 NX 套件版本與題目 1. 的任務執行心得.

心得:這次期末考利用老師給的Uarm零件更改，到了絞鍊的時候遇到的問題就比較多了，例如:拉伸的時候很常因為一些而造成拉伸錯誤，只能一步一步慢慢來

2.NX12組裝

3.完成新幾何外型組立的 uArm 機械手臂後, 將組立件轉入 CoppeliaSim 4.3.0 rev3. 後, 套用 https://www.coppeliarobotics.com/helpFiles/en/simUI-widgets.htm 圖形介面, 以 slider 控制前三軸的轉動. 完成後, 請在個人網頁的 "期末考週" 頁面, 加入第二個 Wink 影片展示過程細節說明.

4.完成新幾何外型組立的 CoppeliaSim 場景後, 請與同組組員合作, 透過 https://mde.tw/cd2022_guide/content/Visualization%20Stream.html 模式, 在 Teams 會議中, 以畫面分享及錄影功能, 由兩位 (含) 以上組員測試題目 3. 的 XML UI slider 控制功能.

BG3-XML UI slider 控制功能

心得:需先去w16下載操控的程式，並依上課老師所教，將控制三軸之頁面顯現，即可控制 CoppeliaSim 場景的三軸。

5.接續題目 4. 的畫面分享與影片錄製模式, 由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 加入 suctionPad 零件, 以手動操控或程式操控的方式, 錄製 block 方塊零件的取放模擬影片.

BG3-block 方塊零件的取放模擬

心得:利用w15程式控制方式用在本題，使用鍵盤操控 K、L操作joint1，H、J操作joint2，F、G操作joint3，P、Q操作吸盤吸或放

6.接續題目 5. 的畫面分享與影片錄製模式, 由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 以跨網路 Python remote API 的方式, 錄製圖形介面 slider 控制前三軸的轉動影片.BG3-跨網路 Python remote API

心得:查看協同組員之IPV4，將程式的self.clientID' ..... '中的IPV4改成組員的IP，組員需在 CoppeliaSim 中的 Main script 裡增加一行程式 simRemoteApi.start(19999) ，將控制三軸之頁面顯現，需要按To connect進行連結的動作，就可以控制組員 CoppeliaSim 場景的三軸。
